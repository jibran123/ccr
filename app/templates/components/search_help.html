Search Help & Examples

Search Modes:

  • Simple Search: Just type keywords (e.g., user api)
  • Property Search: Search specific fields
    • APIName=UserService
    • Platform!=AWS
    • Environment contains prod
    • APIVersion > 2.0
    • LastUpdated >= 2024-01-01
  • Logical Operations:
    • Platform=AWS AND Environment=prod
    • APIName contains user OR APIName contains customer
  • Regex Search: Use /pattern/ (e.g., /^user.*v[0-9]+/)

Operators:

Operator Description Example
= Exact match APIName=UserAPI
!= Not equal Platform!=Azure
contains Partial match APIPath contains /users/
startswith Starts with APIName startswith User
endswith Ends with APIPath endswith /list
>, <, >=, <= Comparison APIVersion >= 2.0

Tips:

  • Searches are case-insensitive by default
  • Use quotes for values with spaces: UpdatedBy="John Doe"
  • Combine multiple conditions with AND/OR
  • Field names support typo tolerance (Platform/Plaform, Environment/Enviroment)
